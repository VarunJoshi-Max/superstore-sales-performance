# Superstore Sales & Performance Dashboard

## 📊 Project Overview
This project analyzes sales performance, profitability, customer behavior, and return impact using SQL and Power BI.

## 🛠 Tools Used
- PostgreSQL (Data Analysis & KPI Computation)
- Power BI (Dashboard Visualization)
- Excel (Data Cleaning)

## 📈 Key Insights
- Total Sales: ₹14.92M
- Total Profit: ₹1.52M
- Profit Margin: 10.2%
- Return Rate: 10.4%

## 🔍 Analysis Covered
- Regional revenue distribution
- Product category profitability
- Seasonal sales trend
- Return impact assessment
- Customer concentration analysis

## 📁 Repository Structure
- `/SQL` → Database creation and analysis queries
- `/PowerBI` → Interactive dashboard file
- `/Report` → Business performance report
- `/Dataset` → Cleaned dataset

## 🎯 Business Outcome
The analysis identifies Technology as the most profitable category and highlights a 10.4% return rate impacting revenue. Strategic recommendations were provided to improve profitability and operational efficiency.
